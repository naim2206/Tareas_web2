VAN

2022

Lista de tareas con calendario
  • Se creó una aplicación web utilizando HTML y Bootstrap para el Frontend y Node.js y Express para Backend y se utilizó MongoDB para almacenar la información
  • Trabajé con un equipo y usamos GIT para el control de versiones y Render para el deployment
  • Se implementaron las operaciones CRUD para usuarios, tareas y notificaciones
  • Se crearon filtros para las tareas (fecha, nombre, etiqueta, etc.) y se diseñó un calendario para filtrar por fecha

Juego de Ajedrez

2021

Juego de Ajedrez con una IA enemiga
  • Se desarrolló un juego de ajedrez en C completamente funcional utilizando raylib para las interfaces gráficas
  • Se utilizó GIT para el control de versiones para trabajar con el equipo
  • Se implementó un botón de deshacer utilizando un stack y una opción para guardar y cargar el juego desde un archive externo
  • El programa implementa una IA que realiza movimientos evaluados

API de videojuegos

2021

API que permite al programador crear videojuegos
  • Se creó una API en java que permite crear objetos colisionables que siguen las leyes de la física
  • Permite al programador crear ventanas y mostrar objetos fácilmente en la pantalla
  • La librería de clases fue diseñada en UML
  • Implementado utilizando el paradigma orientado a objetos de una forma modular y escalable
